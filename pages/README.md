# Image + Text to Cinematic Video Generator

This project is a web application that converts an image and a text prompt into a cinematic video using the Groq API.

## Features

- Upload an image file.
- Enter a text prompt describing the cinematic scene.
- Generate a cinematic video using Groq AI API.
- Preview or download the generated video.
